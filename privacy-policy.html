Last Updated: February 2026

This Privacy Policy describes how the “Prometheus Hunk” creator brand (“we”, “us”, “our”) handles information when interacting with our tools, website, and connected third-party services such as Pinterest, YouTube, Telegram, X (Twitter), and others.

1. Personal Data Collected

This site does not collect personal information from visitors. For API integrations, we only access data strictly required for:

  • Publishing creator-owned content

We do not collect or store data from other Pinterest users or external accounts.

2. How API Data Is Used

Our integrations only use the Pinterest API to:

  • Create Pins

No automated mass posting, scraping, advertising automation, or multi-user access is performed.

3. Data Sharing

We do not sell, share, or transfer user data to any third party. All access tokens are used only for the creator’s own Pinterest account.

4. Cookies & Tracking

This site does not use cookies or analytics.

5. Security

Access tokens (if stored) are protected and never exposed publicly. We follow best practices for app and token security.
